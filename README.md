# B站关注 → 抖音半自动同步

油猴脚本：从 **B 站**导出关注列表，在 **抖音网页端**半自动搜索同名 UP 并关注。

**定位**：给抖音冷启动兴趣种子（测试推荐流），不是官方跨平台同步工具。

> 当前能力已基本可用。实战中更常见的问题是 **抖音官方频控 / 验证码**，不是功能缺失。

---

## 功能概览

| 能力 | 说明 |
|------|------|
| B 站抓取 | 登录后分页拉取关注列表，写入油猴存储，可导出 JSON |
| 抖音搜索 | 单标签打开用户搜索，自动切到「用户」Tab |
| 精确匹配 | 默认要求展示昵称与目标 **完全相等**（trim 后 `===`） |
| 关注点击 | 仅在匹配通过后点击；点击前后双重核验 |
| 防验证码 | 成功关注后随机冷却 10–15 秒再处理下一个 |
| 空名跳过 | 名单中空昵称自动跳过，不中断流程 |
| 手动/自动 | 可切换；跑完后可汇总问题项并手动重跑 |

## 使用流程

### B 站侧

1. 浏览器登录 [bilibili.com](https://www.bilibili.com)
2. 任意 B 站页面右下角出现面板
3. 点 **「抓取并开始」**（或「仅抓取」后再「打开抖音」）
4. 等待关注列表抓取完成

### 抖音侧

1. 使用 **同一个浏览器配置** 打开 [www.douyin.com](https://www.douyin.com) 并登录  
2. **只保留 1 个**抖音标签（多标签会加重风控）  
3. 面板选择：
   - **自动**：匹配后自动点关注  
   - **手动**：只负责搜索/高亮，你自己点关注后按快捷键  
4. 匹配模式建议保持：**完全精确（字符全等）**  
5. 点 **「开始 / 继续」**

### 快捷键（抖音页，焦点不在输入框时）

| 键 | 作用 |
|----|------|
| `F` | 已关注 → 下一个 |
| `S` | 跳过 |
| `X` | 未找到 |
| `P` | 暂停 |

## 匹配逻辑（精确模式）

脚本默认 **精确匹配**，不是“搜到第一个就点”。

```text
打开该 UP 的用户搜索页
  → 切到「用户」Tab
  → 在页面 DOM 中找「文本完全等于目标昵称」的节点
  → 从该节点定位用户卡上的「关注 / 已关注」
  → 校验卡片展示昵称与目标完全一致
  → 通过才点击
  → 点击后再核验是否变为已关注
  → 成功后随机冷却 10–15 秒，再处理下一个
```

### 精确模式规则

- 展示昵称与目标必须 **完全相等**
- **禁止** 部分包含、模糊相似
- **禁止** 仅因抖音号相同而关注
- **禁止** 横向点到邻卡的关注按钮
- 多个完全同名时，取页面 **最靠上**（更靠前）的结果
- 空昵称自动跳过，不中断流水线

### 日志怎么读

```text
当前匹配模式=exact（非 loose 一律按完全精确）
核验#1 精确文本命中=2 原因=dom_exact_nickname 按钮=关注
点击前复核通过：展示名完全匹配…
关注成功：某某UP
关注成功，冷却 12.4 秒再跳下一个（防验证码）…
```

| 日志 | 含义 |
|------|------|
| `精确文本命中=N` | 页面里有 N 处完全等于目标名的文本 |
| `dom_exact_nickname` | 按展示昵称精确锁定 |
| `拒绝点击：…` | 校验未通过，不会点关注 |
| `空昵称，自动跳过` | 名单脏数据，已跳过 |
| `关注后冷却` | 成功后的随机等待，用于降验证码 |

## 官方风控说明（重点）

功能跑通后，**更常见的是抖音侧频控/验证码**，而不是脚本逻辑错误。

### 容易触发的行为

- 短时间连续关注
- 成功路径过快地跳到下一个
- 多标签同时跑
- 未登录 / 弱环境频繁搜索

### 脚本已做的缓解

- 单标签跳转（固定窗口名，避免狂开标签）
- 关注成功后 **10–15 秒随机冷却**
- 搜索间隔默认约 12 秒（可调，最低 8 秒）
- 跳转过多时短冷却（约 45 秒），不再锁死 5 分钟

### 建议用法

1. 先过验证码，再继续  
2. **只开一个**抖音标签  
3. 间隔不要压太低；宁可慢  
4. 可先自动跑一批，问题项再手动清  
5. 一天不要硬肝过多；被风控就停、改天再跑

## 面板选项

| 选项 | 建议 |
|------|------|
| 自动 / 手动 | 先自动，问题多再切手动 |
| 匹配模式 | 保持「完全精确」 |
| 间隔 ms | 默认 12000；仍触发验证可调到 15000–20000 |

## 数据与存储

- 关注名单、进度存在 Tampermonkey 的 `GM_setValue` 中  
- 同一浏览器配置内，B 站页与抖音页共享  
- 可下载 JSON 备份 / 在抖音面板粘贴导入  

---

## 已知限制

1. **跨平台没有官方 API 可一键迁移**  
2. 同名 UP、改名、未开通抖音号会导致匹配失败（属正常）  
3. 抖音网页 DOM 变更可能导致选择器失效，需要更新脚本  
4. 收藏合集无法直接迁移，只能迁移「人」或主题兴趣  
5. 违反平台规则或过度自动化有封号风险；请仅用于自己的账号、控制节奏
